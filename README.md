##  Problem Statement
Many people hesitate to talk about mental health issues due to stigma, lack of access, or unavailability of immediate support. Professional help is not always instantly accessible.

## Solution
AI Therapist is a web-based mental health support website that provides users with a conversational AI chatbot. It offers supportive responses, maintains chat history, and tracks user mood/performance in a simple and user-friendly interface.

This project is for **educational and supportive purposes only** and does not replace professional medical advice.

---

## Key Features
-  AI-powered therapy chatbot
-  Chat history storage
-  Mood / performance tracker
-  Simple and responsive web UI
-  Emotion Detection

## Target Users
- College students
- Working professionals
- People needing immediate emotional support
